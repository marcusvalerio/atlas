# ATLAS

App de treino editorial e minimalista — registro de séries, timer de descanso, cronômetro de sessão e progresso, com estética inspirada em Apple, Aesop, Off-White, COS, Acne Studios, Notion, Porsche e Muji.

Estático (HTML/CSS/JS puro), sem build step. Deploy direto no Vercel.

Dados de exercícios via [ExerciseDB API (open source)](https://oss.exercisedb.dev).
